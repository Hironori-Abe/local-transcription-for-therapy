# Local Transcription for Therapy (LoTT) v0.9.3

臨床心理・カウンセリング会話のための、ローカル完結の日本語文字起こしデスクトップアプリです。
文字起こし・話者分離・文章校正を、会話データを PC の外へ送ることなく実行できます。

## ダウンロード

| ファイル | 対象 | 備考 |
| --- | --- | --- |
| `LoTT-v0.9.3-windows-x64-cuda-setup.exe` | NVIDIA GPU (CUDA 12.x) | 主配布・安定版 |
| `LoTT-v0.9.3-windows-x64-editor-setup.exe` | GPU 不要 | 校正中心の軽量版（文字起こし・話者分離なし） |
| `SHA256SUMS.txt` | — | 各ファイルの SHA-256 チェックサム |

ダウンロード後の検証（任意）:

```sh
sha256sum -c SHA256SUMS.txt
```

## 変更点

v0.9.2 以降のメンテナンス・改善リリースです。既定の動作（日本語の文字起こし・Gemma 4 E4B による校正）は従来どおりで、設定を変えなければ v0.9.2 と同じように使えます。

### 変更・改善

- **文字起こしの進捗表示を滑らかに** — バックエンドからの離散的な進捗イベントの間を、推定所要時間に基づいて滑らかに補間して表示するようにしました（表示専用で、文字起こし処理の速度・内容には一切影響しません）。
- **全体校正カードの高さを調整** — 内容に応じて伸縮する（最大 85vh）ようにし、長い校正結果でも見やすくしました。

### セットアップ / モデル取得の改善

- **ダウンロード失敗時の案内を分かりやすく** — 話者分離モデルのダウンロードが失敗したとき、原因（ネットワーク不通・利用条件の同意反映待ち〈403〉・トークン不正〈401〉・アクセス権不足〈404〉・レート制限〈429〉・ディスク容量不足 など）に応じて、次に何をすればよいかを日本語で具体的に案内するようにしました。
- **Hugging Face トークンの形式チェック** — 送信前にトークンの形式（`hf_` で始まるか、空白・全角文字の混入、長さ）を検証し、打ち間違いを分かりやすく知らせます。
- **話者分離モデルの完全性判定を強化** — ダウンロードが途中で中断した際に残るマーカーを検出し、一部だけ揃った状態を正しく「未完了」と判定するようにしました。中断後はセットアップタブから不足分の補完ダウンロードを促します。

### 開発・実験的構成向け（Windows CUDA / Editor 版には影響しません）

- **同梱 Lemonade を撤去** — AMD（ROCm / Vulkan）での LLM 校正を、Lemonade デーモン / CLI 経由から llama.cpp `llama-server` の直起動へ統一しました。E4B・12B の両方で ROCm 優先 →（不可時）Vulkan フォールバックで動作します。CUDA 版は従来どおり同梱の CUDA llama-server を使うため影響ありません。
- **llama.cpp バックエンドのダウンロード対応** — AMD 用の `llama-server`（ROCm / Vulkan / CPU ビルド）を GitHub Releases から取得する仕組み（`download_llama_backend_cli.py`）を追加しました。取得元は公式リリースのバイナリのみで、会話・音声データの外部送信は一切ありません。
- **Ubuntu 用 Docker ランチャースクリプトを追加** — AMD GPU / Tauri GUI の開発テスト用（`scripts/run-dev-docker-ubuntu.sh`）。

## 動作要件

- Windows 10 / 11 (x64)
- NVIDIA GPU + CUDA 12.x + cuDNN 9.x（CUDA 版。Editor 版は GPU 不要）
- VRAM 8GB 以上（CUDA 版。CPU のみでの動作は想定していません）
- ディスク空き容量: インストーラーのダウンロードは約 400MB（Editor 版は約 6MB）。インストール後のフットプリントは約 1.5GB 以上（同梱の校正エンジン等の展開分）＋モデルダウンロード分（高精度 12B モデルを使う場合は追加で約 7GB）

## インストールと初回セットアップ

1. インストーラーを実行する
2. アプリを起動し、セットアップタブから Python パッケージ・モデルをインストールする
   - **初回セットアップ時のみインターネット接続が必要です**（依存パッケージ・モデルの取得）
   - 話者分離モデルの取得には Hugging Face トークンが必要です（別 PC からのローカルコピーでも配置可能）
3. セットアップ完了後はオフラインで動作します

> **SmartScreen について**: 本インストーラーはコード署名されていないため、初回実行時に
> Windows SmartScreen の警告が表示されることがあります。「詳細情報」→「実行」で続行できます。
> 配布元（本 GitHub Release）からダウンロードしたファイルか、SHA-256 で確認してください。

## プライバシー

- 通常運用時はインターネットに接続しません
- 会話データ・音声データを外部 API へ送信しません
- LLM 校正を含むすべての推論はローカル（loopback）で完結します
- 外部 LLM アプリ（LM Studio / Ollama）との連携は既定で無効です（インストール時のオプトインでのみ有効化可能）

## 既知の問題

- AMD GPU（ROCm）版は experimental です。本リリースには含まれません
- インストーラーはコード署名されていないため SmartScreen 警告が表示されます（上記参照）

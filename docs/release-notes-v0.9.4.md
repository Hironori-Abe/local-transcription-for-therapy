# Local Transcription for Therapy (LoTT) v0.9.4

臨床心理・カウンセリング会話のための、ローカル完結の日本語文字起こしデスクトップアプリです。
文字起こし・話者分離・文章校正を、会話データを PC の外へ送ることなく実行できます。

## ダウンロード

| ファイル | 対象 | 備考 |
| --- | --- | --- |
| `LoTT-v0.9.4-windows-x64-cuda-setup.exe` | NVIDIA GPU (CUDA 12.x) | 主配布・安定版 |
| `LoTT-v0.9.4-windows-x64-editor-setup.exe` | GPU 不要 | 校正中心の軽量版（文字起こし・話者分離なし） |
| `SHA256SUMS.txt` | — | 各ファイルの SHA-256 チェックサム |

ダウンロード後の検証（任意）:

```sh
sha256sum -c SHA256SUMS.txt
```

## 変更点

編集画面に音声を使った2つの新機能（音声入力・区間聞き直し）を追加したリリースです。どちらもオプトインの後付けダウンロード方式で、導入しなければ既定の動作（日本語の文字起こし・Gemma 4 E4B による校正）は v0.9.3 と同じように使えます。

### 新機能

- **音声入力（マイクから挿入候補を生成）** — 編集画面の各行のマイクボタンで最大15秒録音すると、ローカル AI（Gemma 4 E4B + 音声認識モジュール）が聞き取り、編集欄にそのまま挿入できる候補を最大3件提示します。前後の行の文脈は、同じように聞こえる候補の表記（漢字・語尾など）を選ぶためにだけ使われます。全エディション（CUDA / AMD / Editor）で利用でき、CUDA / AMD 版は GPU、Editor 版は CPU で動作します。
- **区間聞き直し（AI による再文字起こしで置き換え候補を生成）** — 各行の編集欄の左側に追加されたボタンで、その行の時間範囲の音声を切り出し、AI が聞き直して「行の内容を置き換える候補」を最大3件提示します。文字起こし結果が怪しい行を、音声を聞き直しながら手で直す代わりに、AI の聞き直し候補から選んで置き換えられます。行が30秒を超える場合は先頭30秒のみを読み取ります（実行時に通知します）。
- **音声入力パック（後付けダウンロード）** — 上記2機能に必要なモデル等は、設定タブの「音声入力パック」からダウンロードします。
  - CUDA / AMD 版: 音声認識モジュール（mmproj、約1GB。校正用 Gemma 4 E4B が未導入の場合は追加で約4GB）
  - Editor 版: 上記に加えて llama.cpp CPU バックエンドと音声切り出し用 ffmpeg（LGPL 構成、約95MB）を含む合計約 5.1GB
- **英語版 README を追加** — リポジトリに `README.en.md` を追加しました（日本語版が引き続き正となります）。

### 改善

- **未導入時の案内を分かりやすく** — 音声入力・区間聞き直しのボタンは、モデル未導入のときも薄いグレーで表示され、カーソルを合わせると「設定タブの音声入力パックからダウンロードしてください」と案内します。
- **編集画面のボタン配置を整理** — 区間聞き直しボタンは編集欄の左側、音声入力（マイク）ボタンは編集欄の右側に配置しました。行下部は行コピー・句点分割の2ボタンを均等配置しています。
- **対象言語を統一** — Whisper と Gemma 4 E4B 音声 ASR の共通12言語に整理しました。Editor 版では「音声入力パック」から選択できます。

### 削除

- **編集欄右側の句読点追加ボタン（`+、` / `+。`）を廃止** — 音声入力ボタンの移動先を確保するための整理です。句読点は編集欄への直接入力で追加できます（AI 句読点付与・単純句読点付与の一括機能は従来どおり利用できます）。

### プライバシー（新機能に関する補足）

- マイクで録音した音声・切り出した音声区間は、いずれも **PC 内のローカル推論エンジン（loopback 限定）だけで処理**され、PC 外へは送信されません。
- 音声入力パックの取得元は Hugging Face（モデル）と GitHub Releases（ffmpeg / llama.cpp バイナリ）のみで、ダウンロードが必要なのはセットアップ時だけです。
- ダウンロードした ffmpeg は Full 版同梱分と同じ LGPL 構成であることを導入時に自動検証します。

### 内部変更

- 旧 Lemonade SDK への参照を除去し、サードパーティライセンス一覧を更新しました。
- Windows でのファイルダウンロードを curl 優先（PowerShell フォールバック）に変更しました。

## 動作要件

- Windows 10 / 11 (x64)
- NVIDIA GPU + CUDA 12.x + cuDNN 9.x（CUDA 版。Editor 版は GPU 不要）
- VRAM 8GB 以上（CUDA 版。CPU のみでの動作は想定していません）
- ディスク空き容量: インストーラーのダウンロードは約 400MB（Editor 版は約 6MB）。インストール後のフットプリントは約 1.5GB 以上（同梱の校正エンジン等の展開分）＋モデルダウンロード分（音声入力パックは約 1〜5GB、高精度 12B モデルを使う場合は追加で約 7GB）

補足: 音声入力・区間聞き直しの AI エンジンは実行のたびに起動し、完了後すぐに終了して VRAM を解放します。校正と同じく、使っていない間に VRAM を占有し続けることはありません。

## インストールと初回セットアップ

1. インストーラーを実行する
2. アプリを起動し、セットアップタブから Python パッケージ・モデルをインストールする
   - **初回セットアップ時のみインターネット接続が必要です**（依存パッケージ・モデルの取得）
   - 話者分離モデルの取得には Hugging Face トークンが必要です（別 PC からのローカルコピーでも配置可能）
   - 音声入力・区間聞き直しを使う場合は、設定タブの「音声入力パック」もダウンロードします（任意）
3. セットアップ完了後はオフラインで動作します

> **SmartScreen について**: 本インストーラーはコード署名されていないため、初回実行時に
> Windows SmartScreen の警告が表示されることがあります。「詳細情報」→「実行」で続行できます。
> 配布元（本 GitHub Release）からダウンロードしたファイルか、SHA-256 で確認してください。

## プライバシー

- 通常運用時はインターネットに接続しません
- 会話データ・音声データを PC 外の API へ送信しません
- LLM 校正・音声入力・区間聞き直しを含むすべての推論はローカル（loopback）で完結します
- ローカルAIアプリ（LM Studio / Ollama）との連携は既定で無効です（インストール時のオプトインでのみ有効化可能）

## 既知の問題

- AMD GPU（ROCm）版は experimental です。本リリースには含まれません
- インストーラーはコード署名されていないため SmartScreen 警告が表示されます（上記参照）
- 音声入力・区間聞き直しの候補生成には、録音・区間の長さや環境により数秒〜数十秒かかることがあります
